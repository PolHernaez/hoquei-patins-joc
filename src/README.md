# Hoquei Patins Joc 🏒

**Alumne:** Pol Hernáez  
**Mòdul:** Entorns de Desenvolupament – DAM1  
**Centre:** Escola Pia de Mataró  

Microvideojoc d'hoquei sobre patins per torns desenvolupat en Java amb interfície gràfica Swing.

## Com executar

1. Obre el projecte a VS Code amb l'extensió "Extension Pack for Java"
2. Executa `src/ui/GameWindow.java`

## Estructura

```
src/model/     → classes del model (Jugador, Equip, Partit, AccioJoc)
src/logic/     → lògica de la IA (IAController)
src/ui/        → interfície gràfica Swing (GameWindow, GamePanel)
diagrames/     → diagrames UML (classes i comportament)
docs/          → documentació de les 5 fases
```
